# Logo Assets Directory

This directory is for USFTT logo files.

## Required Files

Place the USFTT logo here:
- **usftt-logo.png** (or .svg) - Main logo file

## How to Add Logo

1. Download the logo from http://usftt.org/ (right-click on header logo)
2. Save it as `usftt-logo.png` in this directory
3. Recommended: Convert to SVG for better quality and file size

## Current Logo URL

The logo can be found at:
http://usftt.org/local/cache-vignettes/L150xH130/siteon0-7a790.jpg

## Once Added

The logo will automatically appear in:
- Desktop navbar
- (Future) Mobile navigation
- (Future) PWA app icon

See `/docs/LOGO_SETUP.md` for complete instructions.
